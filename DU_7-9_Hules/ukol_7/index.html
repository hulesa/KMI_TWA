HTML

Celý dokument je následně obsažen v tagu HTML. Ten je rozdělen do 2 částí a to tagy head a body.

Microsoft Visual Studio Code

Na začátku si obvykle webdesigneři vystačí s editorem od Miscrosoftu se jménem Visual Studio Code.

Doctype

Úplně prvním tagem v HTML dokumentu je doctype. Definujeme jím, že textový soubor je HTML dokumentem.

Jazyk HTML

Jazyk HTML je základem pro tvorbu webů. Sám o sobě je velmi jednoduchý a všechny webové stránky ho využívají. Složitější weby HTML míchají s několika dalšími jazyky.

Tagy

HTML se skládá ze značek (tzv. tagů). Samo o sobě umožňuje hlavně dodávat prvkům (dále elementům) na stránce určitý význam a to je i jeho hlavní úkol. Dříve se používalo i na grafické stylování stránek, ale protože vzniklé weby byly nepřehledné, bylo omezeno pouze na webový obsah.

Editor

K psaní HTML kódu se většinou nepoužívá poznámkový blok, jelikož mu chybí mnoho funkcí. Mezi ně patří zejména přehledné zvýrazňování kódu nebo podpora kódování češtiny.

HTML stránka

HTML stránku je velmi jednoduché vytvořit, je to vlastně jen textový dokument.
